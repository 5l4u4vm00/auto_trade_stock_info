# 技術指標判讀快速指南

## 移動平均線（MA）

均線是技術分析最基礎的工具，反映一段時間內的平均成本。

| 均線 | 週期 | 代表意義 |
|------|------|----------|
| MA5 | 5日 | 週線，極短線趨勢 |
| MA10 | 10日 | 雙週線，短線趨勢 |
| MA20 | 20日 | 月線，中短線趨勢（最常用） |
| MA60 | 60日 | 季線，中長線趨勢 |

**多頭排列**：MA5 > MA10 > MA20 > MA60，股價在所有均線之上 → 強勢
**空頭排列**：MA5 < MA10 < MA20 < MA60，股價在所有均線之下 → 弱勢
**黃金交叉**：短天期均線向上穿越長天期均線 → 買進信號
**死亡交叉**：短天期均線向下穿越長天期均線 → 賣出信號

## RSI 相對強弱指標

衡量一段時間內上漲動能 vs 下跌動能的比率。

| 數值範圍 | 意義 | 操作參考 |
|----------|------|----------|
| > 80 | 極度超買 | 強烈賣出信號 |
| 70-80 | 超買 | 留意回檔風險 |
| 50-70 | 偏多 | 持股續抱 |
| 30-50 | 偏空 | 觀望或減碼 |
| 20-30 | 超賣 | 留意反彈機會 |
| < 20 | 極度超賣 | 強烈買進信號 |

**背離**：價格創新高但 RSI 未創新高 → 頂背離（看空）；價格創新低但 RSI 未創新低 → 底背離（看多）

## MACD

由 DIF（快線）、MACD Signal（慢線）、柱狀圖組成。

- **DIF 上穿 Signal**：黃金交叉 → 買進信號
- **DIF 下穿 Signal**：死亡交叉 → 賣出信號
- **柱狀圖由負轉正**：動能轉多
- **柱狀圖由正轉負**：動能轉空
- **零軸以上的黃金交叉**：更強的買進信號
- **零軸以下的死亡交叉**：更強的賣出信號

## KD 隨機指標

短線超買超賣指標，反應比 RSI 更靈敏。

| 數值範圍 | K 值意義 | 操作參考 |
|----------|----------|----------|
| > 80 | 超買區 | 留意 K 值向下穿 D 值（死亡交叉） |
| 50-80 | 偏多區 | 持股 |
| 20-50 | 偏空區 | 觀望 |
| < 20 | 超賣區 | 留意 K 值向上穿 D 值（黃金交叉） |

**低檔黃金交叉**（K、D 都在 20 以下交叉）比高檔的更可靠。
**高檔鈍化**：KD 值持續在 80 以上不下來，代表極強勢，不宜過早放空。

## 布林通道（Bollinger Bands）

用標準差衡量價格波動範圍。

- **股價觸及上軌**：短期可能回檔，但強勢股會沿著上軌走
- **股價觸及下軌**：短期可能反彈
- **通道收窄**：波動率降低，即將出現大幅變動（方向不確定）
- **通道擴張**：波動率增加，趨勢正在發展

## 量能指標

成交量是價格變動的燃料，量價關係是判斷趨勢是否健康的關鍵。

| 量價關係 | 意義 |
|----------|------|
| 價漲量增 | 健康的上漲，趨勢可信 |
| 價漲量縮 | 上漲動能不足，可能即將回檔 |
| 價跌量增 | 恐慌性賣壓或主力出貨 |
| 價跌量縮 | 賣壓減輕，可能接近底部 |
| 爆量（量比>2） | 有重大事件或主力進出 |
| 極度萎縮（量比<0.3） | 市場觀望，可能醞釀變盤 |

## 綜合研判原則

1. **多指標共振**：當多個指標同時發出相同方向的信號時，可信度更高
2. **大週期優先**：月線、季線的方向比日線更重要
3. **量先價行**：成交量的變化往往領先於價格變化
4. **不追高不殺低**：等回測支撐或突破確認再進場
5. **停損紀律**：任何交易都必須設定停損價位

## 風險提醒

- 技術指標是落後指標，反映的是過去的價格行為
- 任何單一指標都可能產生假信號
- 突發的基本面消息可能使技術分析失效
- 小型股、低成交量個股的技術指標可靠度較低
- 本分析僅供參考，不構成投資建議
